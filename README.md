#### 关于mangophp：

Mangophp是一个具有表达力，简单的restful框架

- 简单，快速的路由引擎。
- 强大的依赖注入容器。
- 会话和高速缓存存储的多个后端。
- 表现力直观的数据库ORM。
- 数据库不可知的模式迁移。
- 强大的后台作业处理。
